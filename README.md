**以用户为中心的设计**

- 以用户为中心设计的关键特征是：与用户直接互动

- 以用户为中心设计的主要特征是：research、understand、interact、observation、evaluation

- 以用户为中心设计的主要重点是：解决方案对实际用户（real users）的适用性（suitability）在实际使用情境（real contexts of use）下
